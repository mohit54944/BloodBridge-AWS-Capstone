About BloodBridge

Welcome to BloodBridge, a blood donation management platform designed to streamline the process of connecting blood donors, recipients, and administrators through a secure and user-friendly system.

Our Mission

To ensure timely access to blood resources by leveraging technology to efficiently manage donor information, blood requests, and administrative workflows.

Platform Overview

BloodBridge provides a centralized digital solution where users can register, authenticate securely, and access role-based dashboards. The system is built with scalability and cloud-readiness in mind.

Technology Stack

  • Frontend: HTML, CSS, and JavaScript for clean and responsive user interfaces.
  • Backend: Python (Flask) for handling authentication, routing, and business logic.
  • Data Layer: In-memory storage during development, with scope for database integration.
  • Cloud: AWS services to support scalable and reliable deployment.

Cloud & AWS Services

The application is designed to be deployed on Amazon Web Services (AWS), enabling high availability and future scalability. Key services include:

  • Amazon EC2: For hosting and running the Flask application.
  • DynamoDB: Planned NoSQL database for storing user and request data.
  • IAM: For secure access management between AWS services.
  • Amazon SNS: For sending notifications related to blood requests and updates.

Academic Purpose

BloodBridge is developed as part of an academic capstone project to demonstrate full-stack application development, authentication mechanisms, and cloud-based deployment using AWS.
